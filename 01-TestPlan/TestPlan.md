# 1. Objective

The objective of this testing activity is to verify the functionality of selected game features and document gameplay defects found during manual testing.

This project focuses on demonstrating practical QA documentation skills rather than providing complete game coverage.

---

# 2. Scope

## In Scope

### Functional Testing

- Login
- Hero Selection

### Bug Investigation

- Vision System
- Equipment
- EVO Progression
- Peak Tournament

---

## Out of Scope

- Payment System
- Performance Testing
- Security Testing
- Server Load Testing
- Anti-Cheat System
- Localization Review

# 6. Entry Criteria

Testing starts when:

- Game installation completed
- Test account available
- Internet connection available
- Required all necessary resources downloaded

---

# 7. Exit Criteria

Testing is completed when:

- All planned Test Cases have been executed.
- All discovered defects have been documented.
- Test Summary Report has been completed.

---

# 8. Risks

- Server maintenance
- Temporary game events
- Network instability
- Version updates during testing
